# flask-calculator
## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript 
## 🚀 Features
* **Asynchronous Calculations:** Uses JavaScript JSON formatting to process calculations instantly without refreshing the webpage.
* **Basic Arithmetic:** Supports addition, subtraction, multiplication, and division.
* **Responsive UI:** A clean, modern interface designed with HTML and CSS.
